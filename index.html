RESPONSIVE

I have gained experience through projects that I have done with clients. I have learned fundamentals through courses in my current program and through working with clients on their websites.

PROTOTYPING

Prototyping allows UX designers to identify the ele- ments that will be displayed on the page of the screen. It also allows me to change around elements when doing a usability testing on a product.

INTERACTION

Interaction design is an important aspect of being an user experience designer because this is how users interact with your product. To produce a good interaction is to create an effective user journey that allows the user to flow through the product fluidly.

STORYTELLING

Storytelling Is an important aspect that is necessary in any company's brand. Storytelling allows companies to connect with the audience on an emotional level. This can be done through a collection of written content, videos, pictures, or a mixture of them.
